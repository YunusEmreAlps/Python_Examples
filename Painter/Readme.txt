# Pen Move
-> Up    : 'Up'    or w (lower case)
-> Down  : 'Down'  or s (lower case)
-> Right : 'Right' or d (lower case)
-> Left  : 'Left'  or a (lower case)

# Pen Color
-> Black : 1
-> White : 2
-> Red   : 3
-> Blue  : 4
-> Green : 5

# Basic shape
-> Square : 6
-> Circle : 7
-> Triangle : 8

-> penup and pendown
